Privacy Policy

Last updated: March 15, 2026

Warehouse Spending Tracker is a browser extension that analyzes purchase history from warehouse retail websites. Your privacy is important to us.

Data Collection

We do NOT collect, store, or transmit any personal data to external servers. Specifically:

  • No personal information is collected (name, email, address, etc.)
  • No purchase data is sent to any server
  • No analytics or tracking scripts are used
  • No cookies are set by this extension
  • No browsing history is accessed beyond the specific pages needed for analysis

How Data Is Processed

All purchase data analysis is performed entirely locally within your browser:

  • Receipt data is read from the retailer's website while you are logged in
  • Data is stored in your browser's local storage (chrome.storage.local)
  • Analysis, statistics, and exports are generated on your device
  • Share card images are rendered locally using the HTML Canvas API

Permissions

This extension requests the following permissions:

  • storage — To save scraped receipt data and user preferences locally
  • activeTab — To interact with the current tab when you initiate a scan
  • tabs — To open the results page in a new tab
  • Host permission (costco.com) — To read receipt data from the Costco Orders & Purchases page

Pro License

License key validation is performed entirely offline using a client-side checksum algorithm. No license server or external validation is used. Your license key is stored locally in chrome.storage.local.

Third-Party Services

This extension does not use any third-party analytics, advertising, or data processing services.

Data Deletion

You can delete all stored data at any time by:

  • Uninstalling the extension (all local data is automatically removed)
  • Clearing your browser's extension storage

Disclaimer

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date above.

Contact

If you have any questions about this privacy policy, please open an issue on our GitHub repository or contact us through the Chrome Web Store listing.
